**CodeSeed Club**
CodeSeed Club is a beginner‑friendly programming community created by Saketh Chodisetti, a high school student who enjoys coding and teaching others. The club helps students start learning computer science through simple, approachable courses.

**Sign Up**
Students can join by filling out the sign‑in form and choosing the courses they want to learn.

**Courses Offered**

*Python — A friendly introduction to programming.

*Basics of Object‑Oriented Programming (OOP) — Learn classes, objects, and structured code.

*Scratch for Beginners — A visual way to learn programming logic and create simple games.

*HTML Basics — Learn how websites are built using basic tags and structure.

**About the Club**
CodeSeed Club is designed for beginners of all ages. No experience is required—just curiosity and a willingness to learn.
